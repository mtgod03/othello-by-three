# Othello by Three

> [!NOTE]
> このリポジトリは現在開発中です

3人で対戦できるオセロ

## 特徴

- **効率的なアルゴリズム**：ビットボードを用いて効率的なオセロの演算を実現しています
- **整備された開発環境**： モノレポやTypeScriptなどを活用し、開発のしやすい環境を整備しています
